Willkommen bei OS X Server

OS X Server vereinfacht die Konfiguration, Bereitstellung und Verwaltung von Websites. Die intuitive Benutzeroberfläche erleichtert das Erstellen einer Website und bietet professionellen Webmastern leistungsstarke Funktionen.

Um eigene Webinhalte zu verwenden, können Sie diese Platzhalterseite ersetzen, indem Sie Ihre eigene Indexseite in den Ordner „Sitedateien sichern in“ (DocumentRoot) für diese eigene Website legen.
